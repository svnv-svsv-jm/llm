# LLM's: RAGs and more

This repository shows you how one can locally build a RAG from some documents in a folder for a pre-trained LLM.

## Contributing

See [here](./contributing.md).

## Pre-requisites

- Python and a virtual environment
- `poetry`
- `make`

## Installation

```bash
make install
```
